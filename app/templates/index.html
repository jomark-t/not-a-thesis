Web-based ADNU NOCS Data Center Environment Monitoring and Alarm System

This is a web server dedicated to the Ateneo de Naga University (ADNU) Network Operations and Computer Services (NOCS) Data Center for monitoring the temperature and humidity of ambient area, detecting water and smoke occurences, and watching the server room live using a camera that is also capable of panning and tilting.
